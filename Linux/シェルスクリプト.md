# シェルスクリプト
## シェルスクリプトとは
**コマンドを一括で実行するために.shファイルにまとめること**

## 実行例
### .shファイルの作成
```
vim test.sh
```
(該当ファイルがなければ自動的に作成される)
### .shファイルの中身
```bash
#!/bin/bash
#↑bashシェルでの実行を指定 #!(shebangシェバン)はOSカーネルに対して命令している、かなり大事と思われ
# "#"単体はコメントアウト

#Helloをプロンプトに表示
echo 'Hello!'

#プロンプトで使うコマンドも使える
pwd
ls
```
### 実行
- ファイル権限を実行可能(+x)にする
```
chmod +x test.sh
```
r = read,w =write,x = exe(実行)
- 実行
```
./test.sh
```

## 変数とargs(引数)
### 他の言語と同じように変数、引数も扱える
```bash
#/bin/bash

#変数宣言
var='Hello'

#変数を利用する際は$をつける
echo $var

#list(配列)数字も文字列も入れられる
list=(1 100 "a")

#インデックスは0から、全ての要素を呼び出す際は@
echo ${list[0]}
echo ${list[@]}

#$1...により、プロンプトから引数として渡すことができる、$#は引数の数
echo arg1:$1
echo arg2:$2
echo arg3:$3
echo $#
```

## case(条件分岐)とfor(繰り返し)
### 他言語と同じように条件分岐、繰り返しも扱える
```bash
#!/bin/bash
#for文 
for var in x y z
do
	echo $var
done

#if文(case) "|"はor "*"はelse if 分岐末尾にはesac(caseの逆文字)
case "$1" in
	True | T)
		echo "True"
		;;
	False | F)
		echo "False"
		;;
	*)
		echo "None"
		;;
esac
```
