# Linux基本コマンド
##update関連
```
sudo apt update
```
```
sudo apt upgadable
```
```
sudo apt upgrade
```
パッケージのアップデート、アップグレードができる
## ディレクトリ操作
### カレントディレクトリ内のファイル・ディレクトリの表示
```
ls
```
- ファイル権限の表示
```
ls -l
```
- 隠しファイルの表示
```
ls -a
```
### カレントディレクトリの移動
絶対・相対パスどちらも使える
```
cd bin/
```
- 一つ上のディレクトリに移動
```
cd ..
```
- 一番上のディレクトリ(root)に移動
```
cd /
```
- ホームディレクトリ(/home/user)に移動
```
cd ~
```
## ファイル操作のコマンド
### ファイルの作成
```
touch test.txt
```
### 編集
```
vim test.txt
```
Esc ノーマルモード  
I 編集モード  
: コマンドモード
- wq・・・上書き保存
- q!・・・強制終了
### コピー
```
cp test.txt test2.txt
```
### 削除
```
rm test2.txt
```
### 移動(同一ディレクトリ内に移動で名前変更も可能)
```
mv ~/test2.txt ~/test3.txt
```
## パイプライン
### 二つ以上のコマンドを連結できる(右から左へ結果を渡す)
lsの結果をgrep(検索)に渡す
```
ls | grep "検索ワード"
```
printenv1(環境変数)の出力をgrepに渡し、tail(末尾フィルタ)に渡す
```
printenv | grep "bin" | tail 
```
## リダイレクト
### 出力結果の書き込み
- lsの出力結果をtxtに書き込む
```
ls > test.txt
```
- lsの出力結果をtxtの末尾に書き込む(追記する)
```
ls >> test.txt
```
- 書き込んだ内容を表示
```
cat test.txt
```
